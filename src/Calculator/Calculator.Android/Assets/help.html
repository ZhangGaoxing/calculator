通用说明

1. 清除操作

CE Clear Error，仅清除当前操作
C Clear，清除所有操作
 退格

2. 内存操作

MS Memory Save，内存存储
MC Memory Clear，清除所有内存
MR Memory Read，内存调用
M+ Memory Add，内存增加
M- Memory Minus，内存减少
M Memory，所有内存

3. 其他操作

↑ 功能切换

标准

顺序优先型计算，支持内存操作与历史记录功能。

科学

运算符优先型计算，支持内存操作与历史记录功能。

1. 切换操作

DEG - RAD 角度 - 弧度 切换
HYP 双曲函数 切换
F-E 科学记数法 转换

2. 功能操作

Exp 科学记数法输入
dms 度 转 度分秒
deg 度分秒 转 度

程序员

运算符优先型计算，支持内存操作功能。值类型为有符号型，最高位为符号位。

1. 键盘操作

 数字键盘
 比特键盘

2. 进制操作

HEX 十六进制
DEC 十进制
OTC 八进制
BIN 二进制

3. 字长操作

QWORD 64位
DWORD 32位
WORD 16位
BYTE 8位

4. 功能操作

Lsh 左移指定位数
Rsh 右移指定位数
RoL 左移一位
RoR 右移一位

统计信息

简单的求和，平均以及标准差。通过 Add 与 Del 按键对数据进行添加删除。

日期计算

计算两日期的间隔天数和添加或减去天数后的日期。

颜色

十六进制颜色与RGB颜色转换
